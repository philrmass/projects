# Projects
Phil Mass's personal projects
Hosted at [https://philrmass.github.io/projects/](https://philrmass.github.io/projects/)

## To build
```
git clone git@github.com:philrmass/projects.git
cd projects
yarn
```

## To run locally
```
yarn dev
```

## To deploy
```
yarn deploy
```

## To lint
```
yarn lint
```
